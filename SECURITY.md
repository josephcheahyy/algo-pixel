# Security Policy

## Supported Versions

Currently, only the latest release of Algo Pixel is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an e-mail to Joseph Cheah at cheahyueyeou@gmail.com. All security vulnerabilities will be promptly addressed.

Please include the following information:
- A description of the vulnerability.
- Steps to reproduce.
- Potential impact.

You should receive a response within 48 hours. If the vulnerability is accepted, we will coordinate a fix and release it as quickly as possible.
